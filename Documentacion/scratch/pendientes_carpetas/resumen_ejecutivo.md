# Estructura: Resumen Ejecutivo

> Archivo destino: `Documentacion/Presentacion_general/resumen_ejecutivo.md`

## Título
- Nombre del proyecto
- Subtítulo
- Fecha
- Versión
- Autor/es

## 1. Problema
- Encabezado: "El problema"
- 1 párrafo: contexto y estadísticas clave
- 1 párrafo: por qué las soluciones existentes no funcionan

## 2. Solución
- Encabezado: "La solución"
- 1 párrafo: qué es el sistema
- 1 párrafo: cómo funciona (sensor LoRa → gateway → servidor → dashboard)

## 3. Tecnología
- Encabezado: "Tecnología"
- Lista de componentes tecnológicos principales
- Ventajas técnicas (sin conectividad, autónomo, tiempo real)

## 4. Modelo de negocio
- Encabezado: "Modelo de negocio"
- Propuesta de valor
- Clientes objetivo
- Estrategia de monetización / comercialización

## 5. Impacto
- Encabezado: "Impacto"
- Beneficios esperados (económicos, sociales, ambientales)
- Métricas clave

## 6. Estado actual
- Encabezado: "Estado actual"
- Fase del proyecto
- Hitos alcanzados
- Próximos pasos

## 7. Contacto
- Información de contacto del equipo
