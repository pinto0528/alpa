# Estructura: One-Pager

> Archivo destino: `Documentacion/Presentacion_general/one_pager.pdf`

## Formato del documento
- Tamaño: A4 o carta
- Orientación: vertical
- Extensión: 1 página (cara y contracara opcional)
- Tipo de archivo: PDF

## Secciones del one-pager (anverso)

### Encabezado
- Logo del proyecto
- Nombre del proyecto
- Eslogan / tagline

### Bloque superior izquierdo: El problema
- Título
- 1 párrafo breve
- Estadística destacada (número grande)

### Bloque superior derecho: La solución
- Título
- 1 párrafo breve
- Icono o ilustración conceptual

### Bloque central: Cómo funciona
- Diagrama de flujo simplificado
  - Recuadro 1: Sensor en campo
  - Recuadro 2: LoRa → Gateway
  - Recuadro 3: Servidor en la nube
  - Recuadro 4: Alerta al productor
- Flechas de conexión entre recuadros

### Bloque inferior izquierdo: Tecnología
- 3-4 bullets con los componentes clave
- Iconos representativos

### Bloque inferior derecho: Impacto
- 3-4 bullets con beneficios medibles
- Indicadores numéricos

### Pie de página
- Web / contacto
- "Proyecto EmprendeU — Universidad ..."

## Secciones (contracara opcional)
### Reverso (si aplica)
- Equipo
- Estado del proyecto
- Próximos hitos
