# Ingresos y Costos

**ALPA — Sistema de Alerta de Incendios Rurales**

**Fecha:** Junio 2026
**Versión:** 1.0

## Fuentes de Ingresos

### 1. Venta de Hardware (One-Time)

El hardware se vende por separado:

| Componente | Precio venta | Costo | Margen |
|---|---|---|---|
| Gateway | USD 200 | ~USD 130 | 35% |
| Nodo sensor | USD 150 | ~USD 110 | 26.7% |

El productor compra 1 gateway + la cantidad de nodos que cubran su campo. Un productor promedio (200-500 ha) adquiere ~6 nodos por gateway.

### 2. Suscripción Mensual de Monitoreo

Acceso a la plataforma de monitoreo con:

- Dashboard en tiempo real
- Historial de eventos
- Alertas automáticas
- Geolocalización de incidentes
- Reportes descargables para seguros

**Precio: USD 200/mes por productor** (independiente de la cantidad de nodos).

Costo estimado de operación: ~USD 10/mes (hosting, mantenimiento). Margen: 95%.

La suscripción es la principal fuente de ingresos recurrentes y el verdadero motor del negocio a largo plazo.

### 3. Licenciamiento para Aseguradoras

Las aseguradoras agrícolas contratan acceso a información auditada del sistema. Beneficios:

- Validación de medidas preventivas
- Historial de funcionamiento del sistema
- Reportes de eventos
- Reducción de siniestralidad

El licenciamiento se cobra de forma anual o por cantidad de establecimientos monitoreados.

### 4. Bonificaciones de Póliza

El sistema permite que las aseguradoras otorguen descuentos en pólizas a productores que instalen la tecnología. Esto funciona como incentivo para aumentar la adopción del producto y generar acuerdos comerciales entre la empresa y las aseguradoras.

> **Nota:** Las licencias B2B con aseguradoras se encuentran en etapa de validación comercial. No se incluyen en las proyecciones financieras hasta contar con al menos un acuerdo firmado.

## Estructura de Costos

### Costos Fijos Mensuales

Son costos que existen independientemente de la cantidad de equipos vendidos:

| Concepto | Descripción |
|---|---|
| Hosting y servidores cloud | Infraestructura para plataforma de monitoreo |
| Dominio web | Registro y renovación anual |
| Herramientas de desarrollo | Licencias de software, IDE, repositorios |
| Almacenamiento de datos | Base de datos y respaldos |
| Mantenimiento de la plataforma | Actualizaciones, parches de seguridad |
| Espacio de trabajo y administración | Gastos operativos generales |

**Costo fijo mensual estimado: USD 300**

### Costos Variables por Kit (1 Gateway + 1 Nodo)

Para ver el desglose detallado de componentes, ver `costos_produccion.md`.

|---|---|---|
| Hardware | USD 105 | USD 95 |

## Márgenes Estimados

| Concepto | Valor |
|---|---|
| Precio gateway | USD 200 |
| Precio nodo sensor | USD 150 |
| Costo gateway | USD 130 |
| Costo nodo sensor | USD 110 |
| Margen bruto gateway | 35% |
| Margen bruto nodo | 26.7% |
| Margen suscripción (mensual) | 95% |

El margen del hardware es acotado; el negocio se sostiene en las suscripciones recurrentes. Cada cliente que renovás genera ~USD 2.280/año de ganancia neta sin esfuerzo adicional.
